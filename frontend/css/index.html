# 1.1 层叠

层叠决定如何解决样式冲突

  1. 样式来源：浏览器默认样式 < 作者样式 < 作者!important
  2. 样式优先级:行内样式!important > 选择器样式!important > 行内样式 > 选择器样式
  3. 选择器优先级：ID > 类(伪类、属性) > 标签(伪元素)

控制继承：inherit, initial, unset, and revert
