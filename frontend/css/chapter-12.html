# 补间动画

「补间动画」（Tween Animation）指的是：人为设定动画的关键状态（也就是关键帧），而关键帧之间的过渡过程则由计算机处理渲染的动画处理形式。

开发步骤：

  1. 动效审查与分解
  2. 根据需求进行切图
  3. 页面编码
  4. 结合动画属性分解表实现动画

基于 AE 实现 Web 动效：

基于 AE 手工实现 Web 动画的主要工作有两个：

  1. 取参 —— 在动效稿上拿到元素的参数信息，如 x/y/z、rotation、动画设定等
  2. 开发 —— 通过适当的 Web 技术进行动画开发，如 CSS3/Canvas/SVG 等

AE 动画导出为 Web 动画的插件或开源库（代表性的有 Bodymovin 和 lottie-web）

# 逐帧动画

定格动画，又名逐帧动画，是一种动画技术，其原理即将每帧不同的图像连续播放，从而产生动画效果。

  1. 准备动画帧所需的雪碧图
  2. 实现动画
# 使用 CSS3 阶梯函数 steps(number_of_steps, direction)

CSS3 使用 animation-timing-function 的阶梯函数 steps(number_of_steps, direction) 来实现逐帧动画。将动画帧放到 background-image 中,通过改变 background-position 的值来实现动画帧切换。

step-start 与 step-end

除了 steps 函数，animation-timing-function 还有两个与逐帧动画相关的属性值 step-start 与 step-end。

  • step-start 等同于 steps(1, start)：动画执行时以开始端点为开始。
  • step-end 等同于 steps(1, end)：动画执行时以结尾端点为开始。

《CSS 技巧：逐帧动画抖动解决方案》

CSS3 实现的逐帧动画，如果它的帧数较多或尺寸较大时，移动端可能会存在渲染性能问题，此时建议改用 Canvas 实现
