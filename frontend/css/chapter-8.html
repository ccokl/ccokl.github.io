# 1.1 HTML 新语义元素

结构化分组元素：

main:表示页面的主内容

section:用于定义文档或应用中一个通用的区块

nav:用于包装指向其他页面或同一页面中不同部分的主导航链接

article:用于包含一个独立的内容块

aside:元素用于包含与其旁边内容不相关的内容

figure &figcaption:可用于包含注解、图示、照片、代码

detail & summary :“展开”/“收起”部件

header:用在站点页头作为“报头”

footer:用于在相应区块中包含与区块相关的内容，可以包含指向其他文档的链接，或者版权声明

address:用于标记联系人信息

h1到h6: h1到h6元素不能用于标记副标题、字幕、广告语，除非想把它们用作新区块或子区块的标题。

文本级元素：

b:元素表示只为引人注意而标记的文本，不传达更多的重要性信息，也不用于表达其他的愿望或情绪

em:表示内容中需要强调的部分

i:一段文本，用于表示另一种愿望或情绪，或者以突出不同文本形式的方式表达偏离正文的意思

# 1.3 表单

<from autocomplete="off">
  <label for="search">search</label>
	<input id="search" typr="search" placeholder="" required autofocus autocomplete="off" list="awards">
  <datalist id="awards">
    <select>
      <option value="best picture"></option>
    </select>
  </datalist>
</from>

为占位符文本添加样式:input:placeholder-shown {}

required表明该项为必填项

autofocus让表单在加载完成时即有一个表单域被默认选中，以方便用户输入

autocomplete自动补全功能来辅助用户输入

list及对应的datalist元素:可以在用户开始在输入框中输入值的时候，显示一组备选值
