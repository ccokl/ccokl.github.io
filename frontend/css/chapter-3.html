# 3.1 调整盒模型

:root {
  box-sizing:boeder-box
}
*,
::before,
::after{
  box-sizing:inherit
}

使用 inherit 强制继承盒模型，在必要时选中第三方组件的顶级容器恢复成content-box

# 3.2.2 百分比高度

百分比参考的是元素容器块的大小，但容器高度通常由子元素高度决定，因此要想让百分比高度生效，必须给父元素一个明确的高度

等高列实现(现代浏览器)

  1. display:table(显示为table的元素宽度不会扩展到100%；外边距不作用于table-cell)
.wrap {
  margin 0 -1.5em;
}
.container{
  display:table;
  with:100%;
  border-spacing:1.5em 0;
}
.main {
  display:table-cell;
  with:70%;
}
.sider {
  display:table-cell;
  with:30%;
}
  1. Flexbox（默认产生等高元素）
.container{
  display:flex;
}
.main {
  with:70%;
}
.sider {
  with:30%;
  margin-left:1.5em;
}

# 3.3 负外边距

设置左和上负外边框，元素会相应左姨或者上移；设置右或者下负外边距，是将后面的元素拉过来；

# 3.4 外边距折叠

只有上下外边距会折叠

防止折叠：

  • 对容器使用overflow:auto;副作用小
  • 在两个外边距之间加上内边距或者边框
  • 容器为浮动、内联块、决定定位或固定定位
  • Flexbox布局；网格布局
  • table-cell

除非他们能产生想要的布局否则不要轻易使用
