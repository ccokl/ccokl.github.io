# 关于图片资源

Web 图片格式有 JPEG/JPG、PNG、WebP、Base64、SVG 等

# JPEG/JPG

特点：有损压缩、体积小、加载快、不支持透明

适用于呈现色彩丰富的图片，在日常开发中，JPG 图片经常作为大的背景图、轮播图或 Banner 图出现。

# PNG-8 与 PNG-24

特点：无损压缩、质量高、体积大、支持透明

主要用它来呈现小的 Logo、颜色简单且对比强烈的图片或背景等。

# SVG

在线矢量图形库

特点：文本文件、体积小、不失真、兼容性好、性能较差

SVG 应用场景与动画

# Base64

特点：文本文件、依赖编码、小图标解决方案、作为雪碧图的补充而存在

通过对图片进行 Base64 编码，直接将编码结果写入 HTML 或者写入 CSS，从而减少 HTTP 请求的次数；

# WebP

于 2010 年被提出， 是 Google 专为 Web 开发的一种旨在加快图片加载速度的图片格式，它支持有损压缩和无损压缩；

# WebP 的局限性

  • 兼容性
  • 增加服务器的负担
